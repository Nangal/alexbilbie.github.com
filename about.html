Alex Bilbie

I’m a developer at the University of Lincoln working towards the concept of the “connected university” by opening up disparate data stores, adding sexy RESTful APIs on top and then promoting people inside and outside the university to build awesome things with the data.

I developed the Common Web Design HTML5 and CSS3 presentation framework, run the data.lincoln.ac.uk platform, built an OAuth 2.0 powered single sign on platform and have created countless little hacks which bolt disparate systems together.

I’ve worked on 5 JISC funded projects:

  • JISCPress (2009 – 2011)
  • Total ReCal (2010 – 2011)
  • Jerome (2011)
  • Linking You (2011)
  • Linkey (2012 – 2013)

I’ve open sourced a number of pieces of code:

  • CodeIgniter OAuth 2.0 server
  • CodeIgniter MongoDB library

I’ve also contributed to a number of open source projects including CodeIgniter, FuelPHP, Bootstrap and HTML5 boilerplate.

I have spoken at a number of conferences:

  • Dev8D 2011 (workshop – MongoDB)
  • EduServ Federated Access Management Conference 2011 (topic – OAuth 2.0)
  • Dev8D 2012 (workshop – HTML5)
  • CodeIgniter UK Conference 2012 (topic – MongoDB)
  • Dev8ed 2012 (topic – HTML5 & CSS3, The @LNCD Tool Chain)

I have also helped organise a number of conferences:

  • DevXS 2011
  • Dev8D 2012
  • CodeIgniter UK Conference 2012
