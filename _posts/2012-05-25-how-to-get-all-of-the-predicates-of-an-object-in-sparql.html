--- layout: default title: How to get all of the predicates of an object in SPARQL permalink: /2012/05/how-to-get-all-of-the-predicates-of-an-object-in-sparql/ --- I finally discovered the other day how to get all of the predicates of an object in SPARQL. For this example I'll query the University of Cambridge endpoint to get all of the information they have about The Lord of the Rings. 
SELECT * WHERE
{
    'http://data.lib.cam.ac.uk/id/entry/cambrdgedb_1327612' ?p ?o .
}
This query returns the following data:

http://purl.org/dc/terms/identifierUkCU1327612
